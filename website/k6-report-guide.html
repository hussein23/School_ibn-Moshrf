🧪 اختبار أداء موقع المهارات الرقمية

دليل تشغيل k6 وقراءة النتائج

📦 تثبيت k6 على جهازك

🪟 Windows

افتح PowerShell كمسؤول:

winget install k6 --source winget

أو نزّل المثبّت من: dl.k6.io/msi/k6-latest-amd64.msi

🍎 macOS
brew install k6
🐧 Linux (Ubuntu/Debian)
sudo gpg -k
sudo gpg --no-default-keyring --keyring /usr/share/keyrings/k6-archive-keyring.gpg \
    --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys C5AD17C747E3415A3642D57D77C6C491D6AC1D69
echo "deb [signed-by=/usr/share/keyrings/k6-archive-keyring.gpg] https://dl.k6.io/deb stable main" \
    | sudo tee /etc/apt/sources.list.d/k6.list
sudo apt-get update && sudo apt-get install k6

▶️ كيف تشغّل الاختبار

1

تأكد من تثبيت k6

k6 version
2

شغّل الاختبار مع تقرير HTML

k6 run k6-test.js

سيتشغّل 4 سيناريوهات متتالية (إجمالي ~3 دقائق)

3

افتح التقرير (يُنشأ تلقائياً)

start k6-report.html

أو open k6-report.html على macOS

📋 السيناريوهات المبرمجة

السيناريو المستخدمون المدة الهدف
🔥 إحماء تدريجي 0 → 10 20 ث التأكد من الجاهزية
📈 تحميل طبيعي 30 45 ث p95 < 500ms
⚡ ذروة الضغط 60 40 ث p95 < 800ms
🌊 ضغط شديد 100 30 ث p95 < 1500ms

📊 مؤشرات الأداء الرئيسية

p95
95% من الطلبات أسرع من هذا الزمن — الأهم
RPS
عدد الطلبات في الثانية
TTFB
وقت أول بايت من الخادم
Error Rate
نسبة الطلبات الفاشلة

🏆 مقياس التقييم

وقت الاستجابة المتوسط
ممتاز < 200ms جيد < 500ms بطيء > 1000ms
معدل النجاح
ممتاز ≥ 99% جيد ≥ 95% مشكلة < 90%
p95 عند الذروة
ممتاز < 500ms مقبول < 1000ms سيء > 2000ms
